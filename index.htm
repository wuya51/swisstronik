EVM

The Hyperliquid EVM is available on Testnet. It would be great to get initial dev feedback. Please note that it is a rough MVP, testnet only, and may be reset without warning to fix issues that arise. There are some big features that are not yet done, including:

  1. EVM block explorer

  2. JSON-RPC queries with non-latest block number

  3. Integration with other L1 state (In Progress)

The JSON-RPC endpoint is https://api.hyperliquid-testnet.xyz/evm. The chain ID is 998.

Note that there are currently no official frontend components of the EVM. Users can build their own frontends or port over existing EVM applications. All interaction with the EVM happens through the JSON-RPC. For example, users can add the chain to their wallets by entering the RPC URL and chain ID. There is no websocket JSON-RPC endpoint for the EVM.

To use the EVM, get testnet USDC from the faucet and buy native HYPE on the L1. Then, send HYPE to 0x2222222222222222222222222222222222222222 to receive HYPE on the EVM, which is the native gas token. See the instructions in Native Transfers for more details on how this works.
